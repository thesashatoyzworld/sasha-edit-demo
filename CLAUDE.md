# Правила для этого сайта

Это сайт-визитка. Когда тебя просят что-то поменять — меняй **только** текст, цвета, фото и блоки на странице `index.html`.

## Можно
- Менять любой текст (заголовки, описания, кнопки).
- Менять цвета и оформление.
- Добавлять и убирать блоки.

## Нельзя
- Ломать вёрстку — после правки страница должна открываться и выглядеть аккуратно на телефоне и на компьютере.
- Удалять файл `CLAUDE.md`.
- Трогать настройки деплоя.

## Публикация (ВАЖНО)
После каждой правки, которую одобрил пользователь, **сразу отправляй изменения на живой сайт**:
выполни `git add -A`, `git commit -m "<что поменял>"` и `git push`.
Сайт сам обновится через ~30 секунд. Не жди отдельной просьбы «опубликуй» — публикуй после каждой правки.

После пуша скажи пользователю: «Готово, через минуту обнови сайт: https://sasha-edit-demo.vercel.app».

## Как отвечать
Говори по-русски, простыми словами. После правки коротко скажи, что именно поменял.

Если правка может что-то сломать — сначала предупреди.
